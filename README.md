SQL Bar
=======

A bar, in SQL.

Install [PostGres](http://www.postgresql.org/). Glance at the [docs](http://www.postgresql.org/docs/9.4/static/sql.html).

Don't forget to create a database:

    createdb bar


Migrations can be run with:

    psql bar < migrations/0x_xxxx.sql


Happy Hacking!
